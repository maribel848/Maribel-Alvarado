### calculadora simple

```psc
Algoritmo calculadoraSimple
	escribir "que oprecion desea realizar? +,-,*,/"
	leer op
	escribir " ingrese primer numero"
	leer n1
	escribir " ingrese el segundo numero"
	leer n2
	escribir "el resultado de la op es?"
	segun op Hacer
	"+":
			escribir n1 + n2
		"-":
			escribir n1 - n2
		"*":
			escribir n1 * n2
		"/":
			escribir n1 / n2
		De Otro Modo:
			escribir "op no valida"
			
			
			
	FinSegun
```
